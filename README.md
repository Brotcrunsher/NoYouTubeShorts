# Why yet another Implementation?

Existing implementations did too much stuff. Some of them were multiple 100kb in size, while this implementation is only around 2kb. It was too much hassle to read all that stuff so I instead decided to implement this very minimal version.

# How to use
- Download
- Chrome -> Extensions
- Enable Developer mode (top right)
- "Load unpacked" (top left)
- Select the directory of this repo
- Done
